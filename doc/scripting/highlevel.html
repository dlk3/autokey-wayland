def click_on_pat(pat: str, mousebutton: int = 1, offset: (float, float) = None, tolerance: int = 0, restore_pos: bool = False): (source) ¶

Requires C{imagemagick}, C{xautomation}, C{xwd}.

Click on a pattern at a specified offset (x,y) in percent of the pattern dimension. x is the horizontal distance from the top left corner, y is the vertical distance from the top left corner. By default, the offset is (50,50), which means that the center of the pattern will be clicked at.

Parameters
pat:strpath of pattern image (PNG) to click on.
mousebutton:intmouse button number used for the click
offset:(float, float)offset from the top left point of the match. (float,float)
tolerance:intAn integer ≥ 0 to specify the level of tolerance for 'fuzzy' matches. If negative or not convertible to int, raises ValueError.
restore_pos:boolreturn to the initial mouse position after the click.
Raises
PatternNotFoundRaised when the pattern is not found on the screen

def visgrep(scr: str, pat: str, tolerance: int = 0) -> int: (source) ¶

Usage: C{visgrep(scr: str, pat: str, tolerance: int = 0) -> int}

Visual grep of scr for pattern pat.

Requires xautomation (http://hoopajoo.net/projects/xautomation.html).

Usage: C{visgrep("screen.png", "pat.png")}

Parameters
scr:strpath of PNG image to be grepped.
pat:strpath of pattern image (PNG) to look for in scr.
tolerance:intAn integer ≥ 0 to specify the level of tolerance for 'fuzzy' matches.
Returns
intCoordinates of the topleft point of the match, if any. Raises L{PatternNotFound} exception otherwise.
Raises
ValueErrorRaised if tolerance is negative or not convertable to int
PatternNotFoundRaised if C{pat} not found.
FileNotFoundErrorRaised if either file is not found
