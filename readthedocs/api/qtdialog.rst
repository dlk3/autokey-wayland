dialog API when running autokey-qt
==================================

These API methods are available when running the "autokey-qt" front-end.

This class is invoked as the "dialog" class in AutoKey scripts.  For example, the
"autokey.scripting.dialog_qt.QtDialog.calendar()" method documented below is
called as "dialog.calendar()" in an AutoKey script.
