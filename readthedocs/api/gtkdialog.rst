dialog API when running autokey-gtk
===================================

These API methods are available when running the "autokey-gtk" front-end.

This class is invoked as the "dialog" class in AutoKey scripts.  For example, the
"autokey.scripting.dialog_gtk.GtkDialog.calendar()" method documented below is
called as "dialog.calendar()" in an AutoKey script.
