window API implementation for GNOME/Wayland environments
========================================================

This class is invoked as the "window" class in AutoKey scripts.  For example, the
"autokey.scripting.window_gnome.Window.activate()" method documented below is
called as "window.activate()" in an AutoKey script.

There are two different implementations of the "window" class, one for GNOME/Wayland environments one for X11.  This is the GNOME/Wayland version.
